# 華山 Arita-yaki Demo

華山授权海外销售伙伴的多语言前台 Demo。项目面向海外餐厅主厨与采购者，以询价为主要转化方式，并通过器物介绍、产地文化与 Condition Report 呈现有田烧的价值。

## Demo 范围

- 日语默认，支持英语与繁体中文
- 台湾、澳大利亚、美国市场与演示币种
- 器物目录、商品详情、文化介绍、瑕疵说明与询价流程
- 纯前台演示，商品信息、价格、运费和合作方资料均为占位内容

## 本地运行

```bash
pnpm install
pnpm dev
```

## 验证

```bash
pnpm test
pnpm build
```
